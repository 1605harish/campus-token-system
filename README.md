# Campus Token Reward System

## Overview
A blockchain-based reward system for campus activities where students earn "Campus Coins" for participation in events, volunteering, and clubs.

## Features
- ERC-20 based token (CampusCoin)
- Organizer-based reward distribution
- Duplicate reward prevention
- Marketplace for redeeming rewards
- Transparent and tamper-proof ledger

## Contracts
1. CampusCoin.sol - Token contract
2. CampusRewards.sol - Reward distribution
3. CampusMarketplace.sol - Redemption system

## Workflow
1. Admin deploys contracts
2. Organizers are authorized
3. Students receive tokens for events
4. Tokens can be redeemed for rewards

## Tech Stack
- Solidity
- OpenZeppelin
- Ethereum Smart Contracts

## Consensus
Proof of Authority (PoA) for controlled campus environment
